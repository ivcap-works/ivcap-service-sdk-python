   .. py:method:: write_artifact(mime_type: str, name: Optional[str] = None, collection_name: Optional[str] = None, metadata: Optional[Union[ivcap_sdk_service.itypes.MetaDict, Sequence[ivcap_sdk_service.itypes.MetaDict]]] = None, seekable=False, on_close: Optional[OnCloseF] = None) -> IOWritable
      :abstractmethod:

      Returns a IOWritable to create a new artifact. It needs to be closed
      in order to be persisted. If `on_close` is provided it is called with the
      artifactID.

      :param mime_type: _description_
      :type mime_type: str
      :param name: Optional name. Defaults to None.
      :type name: Optional[str], optional
      :param collection_name: Optional collection name. Defaults to None.
      :type collection_name: Optional[str], optional
      :param metadata: Key/value pairs (or list of key/value pairs) to add as metadata. Defaults to {}.
      :type metadata: Optional[MetaDict | List[MetaDict]], optional
      :param seekable: If true, writable should be seekable (needed for NetCDF). Defaults to False.
      :type seekable: bool, optional
      :param on_close: Called with assigned artifact ID. Defaults to None.
      :type on_close: Optional[OnCloseF], optional

      :returns: A file-like object to write deliver artifact content - needs to be closed
      :rtype: IOWritable
