.. py:function:: deliver_data(name: str, data_or_lambda: Union[Any, Callable[[ivcap_sdk_service.cio.io_adapter.IOWritable], None]], mime_type: Union[str, ivcap_sdk_service.itypes.SupportedMimeTypes], collection_name: Optional[str] = None, metadata: Optional[Union[ivcap_sdk_service.itypes.MetaDict, Sequence[ivcap_sdk_service.itypes.MetaDict]]] = None, seekable=False, on_close: Optional[ivcap_sdk_service.cio.io_adapter.OnCloseF] = None) -> str

   Deliver a result of this service

   :param name: A user friendly name
   :type name: str
   :param data_or_lambda: The data to deliver. Either directly or a callback
                          providing a file-like handle to provide the data then.
   :type data_or_lambda: Union[Any, Callable[[IOWritable], None]]
   :param mime_type: The mime type of the data. Anything not starting with 'text' is assumed to be a binary content
   :type mime_type: Union[str, SupportedMimeTypes]
   :param collection_name: Optional collection name. Defaults to None.
   :type collection_name: Optional[str], optional
   :param metadata: Key/value pairs (or list of k/v pairs) to add as metadata. Defaults to None.
   :type metadata: Optional[Union[MetaDict, Sequence[MetaDict]]], optional
   :param seekable: If true, writable should be seekable (needed for NetCDF). Defaults to False.
   :type seekable: bool, optional
   :param on_close: Called with assigned artifact ID. Defaults to None.
   :type on_close: Optional[Callable[[Url]]], optional

   :raises NotImplementedError: Raised when no saver function is defined for 'type'

   :returns: URL of published artifact
